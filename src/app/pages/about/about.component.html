Nossa Missão

O Sistema Anchieta foi desenvolvido com o objetivo de fornecer uma solução robusta e escalável para gestão empresarial. Utilizamos as tecnologias mais modernas do mercado para garantir performance, segurança e uma excelente experiência do usuário.

Principais Características

  • Desenvolvido com Angular 17 e TypeScript
  • Design responsivo e acessível
  • Arquitetura baseada em componentes
  • Sistema de roteamento avançado
  • Integração com APIs RESTful
  • Autenticação e autorização seguras

Arquitetura do Sistema

Componentes

Sistema modular baseado em componentes Angular standalone para máxima flexibilidade.

Roteamento

Navegação SPA com lazy loading e guards para controle de acesso às rotas.

Serviços

Camada de serviços para comunicação com APIs e gestão de estado da aplicação.

Design System

Interface consistente com SCSS, variáveis CSS e componentes reutilizáveis.
